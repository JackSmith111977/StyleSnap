# StyleSnap 前端开发

## 职责
- Next.js 16 功能开发
- Shadcn UI 组件集成
- Tailwind CSS 样式
- 响应式设计

## 技术栈
- Next.js 16 (App Router)
- React 19.2
- TypeScript 5.7+ (strict)
- Tailwind CSS 4.x
- Shadcn UI + Radix UI
- Zustand (状态管理)
- TanStack Query (服务端状态)
