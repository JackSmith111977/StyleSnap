```markdown

> [主题的一句话描述]

## 内容深度要求（L4 专家级）

| 维度 | 要求 |
|------|------|
| 概念定义 | 准确、简洁，引用官方定义 |
| 工作原理 | 深入底层机制，包含流程图/架构图 |
| 源码解析 | 关键特性包含源码级别解析 |
| 代码示例 | 完整可运行，包含注释和说明 |
| 常见误区 | 指出开发者容易误解的地方 |
| 最佳实践 | 给出实际开发中的推荐使用方式 |
| 性能考虑 | 包含性能分析和优化建议 |
| 对比分析 | 与相似方案进行对比 |
