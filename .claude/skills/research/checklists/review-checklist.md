# Review 检查清单

> 在文档写入前必须完成此检查

---

## 一、结构检查

| 检查项 | 说明 | 状态 |
|--------|------|------|
| 章节编号连续 | 1-8 编号无跳跃 | - [ ] |
| 无重复章节 | 没有重复的章节标题 | - [ ] |
| 目录对应 | 目录与实际章节一致 | - [ ] |
| 章节顺序合理 | 子章节在父章节之后 | - [ ] |

### 检查命令

```bash
# 检查章节编号
grep "^## " filename.md

# 检查重复标题
grep "^## " filename.md | sort | uniq -d
```

| 检查项 | 说明 | 状态 |

| Markdown 正确 | 代码块、表格、列表规范 | - [ ] |

| 表格完整 | 有表头分隔线 | - [ ] |

### 常见格式问题

```markdown
<!-- ❌ 错误：代码块无语言 -->
```
const x = 1;
```

<!-- ✅ 正确：标注语言 -->
```javascript
const x = 1;
```

<!-- ❌ 错误：表格无分隔线 -->
| 头 1 | 头 2 |
|------|------|
| 内容 | 内容 |

<!-- ✅ 正确：有分隔线 -->
| 头 1 | 头 2 |
|------|------|
| 内容 | 内容 |
```

## 问题修复指南

| 问题 | 修复方式 |
|------|----------|
| 章节编号不连续 | 重新编号 |
| 目录与实际不符 | 同步目录 |
| 代码块无语言标注 | 添加语言 |
| 引用格式不统一 | 统一格式 |
| 内容过浅 | 补充原理讲解 |
