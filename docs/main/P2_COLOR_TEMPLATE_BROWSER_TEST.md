### TC-002: 下拉菜单打开

**步骤**:
1. 点击/触发模板选择器按钮
2. 检查下拉菜单是否打开

**预期结果**: 下拉菜单打开，显示 4 个模板选项

**测试结果**: ✅ 通过

**注意**: 在 headless 浏览器中，鼠标点击事件需要配合键盘事件（Enter 键）才能可靠触发 Radix UI DropdownMenu。

**实现细节**:
```javascript
// 可靠的触发方式
templateBtn.focus();
templateBtn.dispatchEvent(new KeyboardEvent('keydown', { key: 'Enter', bubbles: true }));
```

## 发现的问题

### 问题 1: 鼠标点击触发不可靠
**描述**: 在 headless 浏览器中，仅使用 `click()` 或 `dispatchEvent(new MouseEvent('click'))` 无法可靠打开 Radix UI DropdownMenu。

**解决方案**: 使用键盘事件（Enter 键）触发：
```javascript
templateBtn.focus();
templateBtn.dispatchEvent(new KeyboardEvent('keydown', { key: 'Enter', bubbles: true }));
```

**影响**: 仅影响自动化测试，不影响真实用户交互。

## UX 设计师评估 (Sally)

**视觉差异评估**:

4 个模板的视觉差异**明显且易于区分**：

1. **经典商务**: 使用主色（蓝色系）作为标题和按钮背景，专业稳重
2. **活力创意**: 使用辅色作为主按钮，强调色作为链接，活泼有创意感
3. **极简主义**: 使用文字色作为标题，主色仅用于按钮，简洁克制
4. **科技现代**: 使用主色作为卡片头部背景，强调色用于交互元素，科技感强

**建议**: 当前 4 个模板的颜色映射差异足够明显，用户能够清晰感知不同模板带来的视觉风格变化。
