### 新增功能详情

**F2.8 风格预览组件**
- **描述**: 在风格详情页用固定尺寸、响应式的预览组件展示该风格应用效果
- **包含组件**: 导航栏、侧边栏、标题、正文、卡片、列表、页脚
- **实现方式**: CSS Variables + CSS Modules
- **数据来源**: `style_design_tokens` 表
- **深色模式**: 混合模式（算法生成 + 手动覆盖）
- **优先级**: P2
